Day 5: Mastering Layouts with CSS

Welcome to Day 5 of our web development journey! Today, we'll explore the display and position properties in CSS to gain better control over the layout and positioning of elements on a webpage.

The display Property

The display property determines how an element is rendered on the page. There are three commonly used values: block, inline, and inline-block.

Example: Block Element

<div class="block-element"></div>

The block element takes up the full width available and starts on a new line.

Example: Inline Element

<span class="inline-element"></span>

The inline element only takes up as much width as necessary and does not start on a new line.

Example: Inline-Block Element

<div class="inline-block-element"></div>

The inline-block element takes up only as much width as necessary, but it allows for setting a height and width.

The position Property

The position property is used to specify the positioning method of an element. The values include static, relative, and absolute.

Example: Position Static

<div class="position-static"></div>

The static position is the default positioning. Elements are positioned according to the normal flow of the document.

Example: Position Relative

<div class="position-relative"></div>

The relative position allows you to move an element relative to its normal position without affecting other elements.

Example: Position Absolute

<div class="position-absolute"></div>

The absolute position positions an element absolutely to its nearest positioned ancestor, or to the initial containing block if there is none.
