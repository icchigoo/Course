Day 6: Mastering Flexbox

Welcome to Day 6! Today, we'll delve into the world of Flexbox, a powerful layout model that makes it easier to design complex, responsive, and flexible layouts.

Understanding Flexbox

Flexbox is a layout model in CSS that allows you to design complex layouts more efficiently. It involves a flex container and flex items.

Flex Container

<div class="flex-container"></div>

The display: flex; property is applied to create a flex container. It defines a flex context for its direct children.

Flex Items

<div class="flex-item"></div>

Elements inside a flex container become flex items. Flex items can be arranged horizontally or vertically based on the container's properties.

Creating Flexible Layouts

Flexbox provides properties to control the layout, alignment, and distribution of elements within a flex container. Let's explore some common properties:

justify-content

.flex-container {

display: flex;

justify-content: space-between;

}

The justify-content property aligns flex items along the main axis. The space-between value evenly distributes items, leaving equal space between them.

align-items

.flex-container {

display: flex;

align-items: center;

}

The align-items property aligns flex items along the cross axis. The center value vertically centers the items.
